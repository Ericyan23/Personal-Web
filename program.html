Notes: Basic Numpy

Basic

Numpy is a powerful Python library for numerical computations. It's widely used for working with arrays and matrices. Here are some fundamental concepts:

import numpy as np
array1 = np.random.randn(x, y)
// Returns an x by y random array, where 'n' denotes a normal distribution

The above code demonstrates how to create a random array with dimensions x by y using Numpy's random module. 

X.ndim // Find the number of dimensions in the array

Use the ndim attribute to find the number of dimensions in a Numpy array. This can be helpful in understanding the structure of your data.

x.shape // Dimensions of the array, e.g., rows x columns

The shape attribute provides information about the dimensions of the array, such as the number of rows and columns.

  • Vectorization: Numpy allows you to perform operations on entire arrays, making calculations more efficient.
  • Element-wise Comparison: When comparing two arrays, Numpy compares each element individually and returns True or False for each element.

Indexing

  • Array indexing in Numpy starts from 0, similar to most programming languages.
  • Changing values in an array slice affects the original array:
    • array = np.arange(10) // Does not include 10
array_slice = array[5:8]
array_slice[1] = 123 
// Changing the value in the slice will also change the original array. This will modify the values in the original array.

    Transposing Arrays

    X.reshape(x, y) // Reshapes the array into x rows and y columns

    You can use the reshape function to change the dimensions of an array. This can be useful when you need to transform data for specific operations.
